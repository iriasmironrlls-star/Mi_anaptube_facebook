# 🔵 FB Snaptube Pro (Web Edition)

Este proyecto permite descargar contenido de **Facebook** (Videos como MP3, Fotos y Audio) directamente a tu celular Android, organizado automáticamente por carpetas de autores en la raíz del almacenamiento.

## 🚀 Características
- **Interfaz Web:** Controla tus descargas desde el navegador.
- **Organización Inteligente:** Crea carpetas automáticas por usuario de Facebook.
- **Estilo Snaptube:** Incluye carátula, artista y título en cada archivo.
- **Sin FFmpeg:** Funciona nativamente en Pydroid 3.

## 🛠️ Instalación en Pydroid 3
1. Instala las dependencias: `pip install flask yt_dlp requests mutagen`
2. Ejecuta `main.py`.
3. Abre en tu navegador: `http://localhost:5000`
